# 📚 Author Books Website

A modern and responsive author portfolio and book showcase website designed to promote books, grow a reader audience, and build a strong online author presence.

---

## ✨ Features

- 📖 Showcase Multiple Books
- 🧑‍💼 Author Biography Section
- ⭐ Book Reviews & Testimonials
- 🛒 Book Purchase Links (Amazon, Goodreads, etc.)
- 📚 Book Series Display
- 📧 Reader Mailing List Subscription
- 🌐 Social Media Integration
- 📱 Fully Responsive Design
- ⚡ Fast Loading & SEO Optimized
- 📝 Blog / News Section
- 🔍 Search & Filter Books
- 🎨 Modern UI/UX Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 / Tailwind CSS
- JavaScript / TypeScript
- React.js / Next.js
- Node.js
- MongoDB / Firebase
- Vercel Deployment

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/author-books-website.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd author-books-website
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

### 5️⃣ Open in Browser

```bash
http://localhost:3000
```
